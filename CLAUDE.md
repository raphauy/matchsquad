## Información Importante del Usuario

### Proceso de Testing
- **El usuario maneja el servidor de desarrollo por su cuenta**
- **NO ejecutar `pnpm run dev` automáticamente**
- **Cuando necesite que pruebe cambios, solo avisar al usuario**
- El usuario prefiere controlar cuándo reiniciar el servidor

### Documentación de Librerías
- **Usar Context7 para obtener documentación actualizada de librerías**
- Context7 proporciona documentación y ejemplos de código actualizados
- Primero usar `resolve-library-id` y luego `get-library-docs`
